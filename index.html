Unit 2: Computer Systems and Networks

A computing device connects to a computer network using agreed-upon protocols. The internet works by splitting messages into packets that travel along different paths, guided by routers. With redundancy and fault tolerance, the system continues working even if some parts fail. The Domain Name System (DNS) translates human-readable names into IP addresses. Finally, the World Wide Web uses HTTP to link pages and media together.
